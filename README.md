## Project Overview

In this project, I worked on developing a pirate intelligent agent using deep Q-learning to solve a maze pathfinding problem. The starter code included the environment, movement logic, and neural network model. My focus was completing the Q-training algorithm, including the training loop, epsilon-greedy action selection, experience replay, and model training. I also restructured the loop to follow the correct deep Q-learning flow and added comments for clarity. In the end, the agent was able to learn and reach the treasure over time while I closely monitored the output.
